# `help` Command

## NAME
`help` — List available commands or show manual for a specific command

## SYNOPSIS
```
help [command]
```

## DESCRIPTION
The `help` command lists all available commands. If a specific command name is provided, it returns the manual for that command (similar to `man`).

Includes both internal and client-side commands.

## EXAMPLES
```
help
→ Lists all available commands

help echo
→ Shows the manual for the 'echo' command
```
