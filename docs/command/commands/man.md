# `man` Command

## NAME
`man` — Show the manual of a given command

## SYNOPSIS
```
man <command>
```

## DESCRIPTION
The `man` command displays the manual (help documentation) for the specified command. It returns usage information, syntax, description, and examples if available.

If the specified command does not exist, an error message is returned.

## EXAMPLES
```
man help
→ [shows the manual for the 'help' command]

man math
→ [shows the manual for the 'math' command]
```
