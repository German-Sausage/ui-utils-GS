# `math` Command

## NAME
`math` ‚Äî Evaluate a mathematical expression

## SYNOPSIS
```
math <expression>
```

## DESCRIPTION
The `math` command evaluates a given mathematical expression and returns the result.

The expression can include basic arithmetic operators (`+`, `-`, `*`, `/`, `^`) and parentheses for grouping. The expression is parsed and calculated using standard mathematical precedence rules.

## EXAMPLES
```
math 3+5
‚Üí Result: 8.0

math (2+3)*4
‚Üí Result: 20.0

math 2^3
‚Üí Result: 8.0
```
