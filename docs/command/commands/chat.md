# `chat` Command

## NAME
`chat` — Send a chat message to the server

## SYNOPSIS
```
chat <message>
```

## DESCRIPTION
The `chat` command sends a chat message to the server. If the message starts with a `/`, it will be treated as a command and sent as such.

## EXAMPLES
```
chat Hello, world!
→ Sends a regular chat message: "Hello, world!"

chat /help
→ Sends a command message: "/help"
```
