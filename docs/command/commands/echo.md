# `echo` Command

## NAME
`echo` — Echo the provided arguments

## SYNOPSIS
```
echo <arg1> <arg2> ...
```

## DESCRIPTION
The `echo` command returns the provided arguments as a single space-separated string. It is primarily used for testing or displaying raw input.

## EXAMPLES
```
echo Hello World
→ Hello World

echo 123 abc
→ 123 abc
```
