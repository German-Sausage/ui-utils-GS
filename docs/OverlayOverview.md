# Overlay Overview

The Ui-Utils Overlay provides different tools preform different client sided actions.
Each widget offers unique functions — from closing screens safely to saving your screen layouts!

This documentation explains what each widget does, how to use it, and links to detailed guides when necessary.

## Quick Notes
- **Keybind**: Press **V** (by default) to restore a saved screen from the "default" slot.
- **Web Terminal**: A terminal-like interface for running commands can be found [here](https://mrbreaknfix.com/terminal), You can execute different commands or read more about it at the [command system documentation page](command/CommandSystem.md).
- **Delay Packets**: If enabled, GUI actions and Chat Box messages are delayed until Delay Packets is set to false.
