# Get Plugins

## What It Does
Attempts to **retrieve the list of server plugins**

This lets you see which plugins a server might be running.

## How to Use
- Click the **Get Plugins** button.
- Wait a few seconds for a response.
- If successful, plugin names will appear and be printed in chat.

## Notes
- May fail if the server has patched the plugin list command.
- Some servers use anti-plugin leak measures.
- This feature was added for users that didn't want to install [Meteor Client](https://meteorclient.com/), which is where this feature was originally inspired from.

## Tips
- Works best on smaller or poorly configured servers.
