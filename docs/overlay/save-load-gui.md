# Save GUI and Load GUI

## What It Does
Allows you to **save the layout** of your current GUI screen and **reload it later**.

Saving the screen creates a snapshot stored in the "default" slot.

## How to Use
- Click **Save GUI** to save the current screen.
- Click **Load GUI** to restore the saved screen.
- Use the **V keybind** (by default) to quickly load the saved screen during gameplay.

## Access via Web Terminal
You can also view, manage, and load saved screens at  
[https://mrbreaknfix.com/terminal](https://mrbreaknfix.com/terminal) using the `screen` command.

## Notes
- Only the latest saved screen is kept in the "default" slot.
- Saving screens does not persist between game restarts.
