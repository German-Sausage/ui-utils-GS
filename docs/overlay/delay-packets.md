# Delay Packets

## What It Does
Toggles whether outgoing GUI and Chat packets are **delayed** or sent immediately.

When enabled, actions (such as GUI changes or Chat Box messages) are saved and only sent later when you disable Delay Packets, or use the **Leave & Send Packets** button.

## How to Use
- Click the **Delay Packets** button to toggle ON/OFF.
- Green/active = packets are delayed.
- Gray/inactive = packets are sent immediately.

## Notes
- Useful for queueing up multiple actions invisibly.
- When you disable Delay Packets, all saved packets will be sent.

## Tips
- If you plan to chat or interact quietly without alerting the server immediately, enable this!
