# Open Terminal

## What It Does
Opens https://mrbreaknfix.com/terminal in your browser, allowing you to easily run commands.

From here, you can type and execute MrBreak commands easily without leaving the game.

## How to Use
- Click the **Open Terminal** button (bottom right of overlay).
- The terminal overlay will appear, ready for input.

## Notes
- Supports auto-completion and command history.

## Tips
- You can quickly access terminal-only features without alt-tabbing!
