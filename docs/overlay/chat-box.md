# Chat Box

## What It Does
Lets you **send chat messages or commands** while a GUI is open, without having to close the screen.

Messages typed here will behave like normal Minecraft chat messages.

## How to Use
- Type a message or command into the **Chat Box**.
- Press Enter or click the Send button.

## Delay Packets Interaction
- If **Delay Packets** is ON, chat messages are saved and sent later when you turn Delay Packets OFF.
- If **Delay Packets** is OFF, chat messages are sent immediately.

## Notes
- Supports both normal chat (`Hi there!`) and commands (`/home`).

## Tips
- Useful for if you want to open a plugin gui via a command while sending a delayed packet at the same time =]
